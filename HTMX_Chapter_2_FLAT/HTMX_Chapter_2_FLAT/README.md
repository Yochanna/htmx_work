# HTMX — Chapter 2 (Port 3002, Flat Layout)

Unzip this folder anywhere, open PowerShell **in this folder**, then run:

```powershell
.\start.ps1
```

That’s it — server starts at http://localhost:3002 and browser opens automatically.

To stop:
```powershell
.\stop.ps1
```

To restart:
```powershell
.estart.ps1
```
